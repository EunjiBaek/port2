REACT

01. Kauction Live

케이옥션 라이브 경매페이지는 온라인에서도 사용자가 편리하게 경매할수 있게 만든 페이지입니다. 기존 라이브경매페이지를 리액트를 공부해서 새롭게 리뉴얼 하자는 제안을 하였습니다. 상태관리를 편리하게 할 수 있는 중앙집합체 역할을 하는 리덕스를 적용하였고, 비동기 작업을 유연하게 할수 있도록 도와주는 redux-saga를 사용하였습니다.

디렉터리 구조는 액션을 만드는함수, 리덕스를 관리하는 곳, saga를 관리하는곳, api를 불러오는곳, 컴포넌트를 감싸고있고 컴포넌트에 데이터를 관리해주는 container폴더, 페이지의 순수 마크업만을 담당하는 components폴더로 분리하였습니다. 상태관리를 편리하게 해주는 리덕스를 통해서 api를 굳이 불러오지 않더라도 ui를 사용자에게 미리 보여줄수 있도록 구성하였습니다.

또한 react-intl(다국어 패키지)를 사용하여 다국어처리까지 완료한 페이지입니다. 실시간 라이브 경매인만큼 응찰을 했을때 응찰처리가 잘 이루어지는지, 응찰에 대한 오류가 발생했을경우, 응찰을 관리자화면에서 취소나 삭제했을 경우 에러처리가 잘 이루어지는지에 대한 고려를 하였습니다.
